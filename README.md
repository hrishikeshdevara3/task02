Advanced Exploratory Data Analysis (EDA) & Feature Engineering

Project Overview:
The project is to clean and preprocess the e-commerce data.Where We need to clean the data for further machine learning algorithms  though preprocessing and also exloring the EDA (Exploratory Data Analysis) methodology for the Analysis and also to explore the new features in the existing dataset.

Objective:
->Clean and preprocess raw data.
->Handle missing values using appropriate imputation techniques.
->Perform comprehensive Exploratory Data Analysis (EDA).
->Engineer new features from existing data.
->Produce a clean dataset suitable for predictive modeling.

Project Workflow:
1.Data collection:
->Data loading from the sources
->Examining the dataset and data types

2.Data pre-processing:
->Removed duplicate records.
->Corrected inconsistent values.
->Standardized column formats.

3.Missing Value Handling
Mean Imputation
Median Imputation
Mode Imputation
KNN Imputation

4.Outlier Detection
->Interquartile Range (IQR)
->Z-Score Method
->Boxplots

5.EDA(exploratory Data Analysis)
->Summary Statistics
->Distribution Analysis
->Correlation Analysis
->Missing Value Analysis
->Category-wise Sales Analysis
->Product Performance

6.Feature Engineering
These engineered features enhance predictive modeling capability.
->Total Revenue
->Order Month
->Order Day
->Order Weekday
